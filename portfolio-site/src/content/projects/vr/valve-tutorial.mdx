## Overview

The VR Valve Tutorial is a practical training module designed to help apprentices and technicians learn industrial valve assembly through direct, hands-on interaction. Built for Meta Quest, it allows users to practise assembly steps in a controlled, risk-free environment before working with physical equipment.

The experience focuses on clarity and repetition. Users are guided through each stage of the assembly process with contextual prompts and visual callouts that stay anchored to the relevant components. This allows learners to understand both *what* they are doing and *why*, without breaking flow.

The module is designed to support hands-on training, not replace it. The aim is to shorten ramp-up time and help learners build confidence before they step onto the workshop floor.

## Key Features
- Hands-on VR assembly with step-by-step interaction  
- Contextual callouts linked directly to components  
- Real-time feedback during assembly  
- Designed for repeatable training sessions  
- Optimised for standalone Meta Quest  
- Refined through user testing

## Technology Stack:
- Engine: Unity
- Language: C#
- XR Framework: Oculus SDK / OpenXR
- Target Platforms: Meta Quest

## Project Goal & Learning Outcomes:
- Support industrial training with VR  
- Reduce onboarding friction for new starters  
- Improve consistency in training delivery  
- Explore practical XR use in industrial education
